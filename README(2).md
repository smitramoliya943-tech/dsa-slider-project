# 🖼️ Image Slider Project

A simple and responsive **Image Slider** created using **HTML, CSS, and JavaScript**.

## 📌 Project Description

This project displays different famous places using an image slider. Users can navigate through the images using the **Previous (←)** and **Next (→)** buttons.

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript

---

## ✨ Features

- Previous & Next Navigation
- Smooth Image Switching
- Responsive Design
- Easy to Understand Code
- Beginner Friendly Project

## ⚙️ How to Run

1. Download or Clone this repository.
2. Open the project folder.
3. Open **index.html** in any web browser.
4. Use the **← Previous** and **→ Next** buttons to view images.
